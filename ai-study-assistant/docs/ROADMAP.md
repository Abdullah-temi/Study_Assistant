# Roadmap

## Milestone 1: Local Skeleton

- Frontend app shell, routing, and MVP screens
- Backend FastAPI app with modular routers
- Local upload folder and placeholder AI responses
- README with local run instructions

## Milestone 2: Real Authentication

- PostgreSQL connection
- User model and migration setup
- Password hashing
- JWT access tokens
- Protected endpoints

## Milestone 3: Document Uploads

- Store document metadata
- Extract text from TXT, PDF, and DOCX files
- Show uploaded documents in the dashboard
- Add validation and useful upload errors

## Milestone 4: RAG Chat

- Split extracted text into chunks
- Generate embeddings
- Store chunks in ChromaDB
- Retrieve relevant chunks per question
- Answer using only uploaded notes

## Milestone 5: Study Tools

- Summary generation
- Flashcard generation
- Quiz generation
- Saved study artifacts per user

## Milestone 6: Production Readiness

- Tests
- Logging and observability
- Deployment configuration
- Security hardening
- Documentation
